Introduction

This page is a beginner-friendly guide to HTML and CSS. It explains the basic ideas of building web pages and styling them.

HTML Basics

HTML (HyperText Markup Language) is the structure of webpage. It uses <tags>to tell the browser what each part of the page.

For example, a heading can be written like this:

<h1>Hello World</h1>
HTML Elements

Some commen elements in HTML are:

  • <p> → Paragraph
  • <a> → Link
  • <img> → Image
  • <div> → Division
CSS Basics

CSS (Cascading Style Sheets) is used to make HTML look nicer. It controls colrs, spacing, fonts and layouts.

Exemple of changing text color:

p {
  color: blue;
}
CSS Selectors

Selectors are used to choose which HTML elements to style. Some common ones are:

  • p → selects all paragraphs
  • #id → selects an element with a specific id
  • .class → selects all elements with that class
Conclusion

HTML gives a webpage structure and CSS makes it look good
